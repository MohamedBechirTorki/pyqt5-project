## Installation de PyQt5

Selon votre système d’exploitation, veuillez installer PyQt5 comme suit :

### Sous Windows :
```bash
pip install pyqt5
```
### Sous Linux (Debian/Ubuntu) :
```bash
sudo apt install python3-pyqt5
```
